# Project Docs

Bu papka `job_aggregator` loyihasining 2026-03-07 holatidagi strukturali hujjatlarini saqlaydi.

## Hujjatlar xaritasi

- `architecture.md` - servis arxitekturasi, komponentlar va parser plugin kontrakti.
- `status.md` - hozirgacha qilingan ishlar, verifikatsiya natijalari va mavjud artefaktlar.
- `operations.md` - lokal ishga tushirish, Telegram auth, Kafka, live listener va helper scriptlar.
- `todo.md` - keyingi bosqichlar va prioritetlangan ishlar ro'yxati.

## Tez kirish

1. Arxitekturani ko'rish uchun `architecture.md`ni oching.
2. Hozirgi tayyor holatni ko'rish uchun `status.md`ni oching.
3. Servisni ishga tushirish uchun `operations.md`dan foydalaning.
4. Keyingi backlog uchun `todo.md`ni oching.
